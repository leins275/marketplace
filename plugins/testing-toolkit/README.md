# testing-toolkit

The ITS testing standard for Python services and web applications. Bundles three skills spanning unit testing, end-to-end testing and live web-app debugging.

## Skills

| Skill | What it does |
| --- | --- |
| `python-testing-patterns` | Comprehensive testing strategies with pytest — fixtures, mocking and test-driven development. |
| `e2e-testing-patterns` | End-to-end testing with Playwright and Cypress — reliable suites and debugging flaky tests. |
| `webapp-testing` | Interact with and test local web apps using Playwright — verify frontend behavior, debug UI, capture screenshots and browser logs. |

## Install

```text
/plugin install testing-toolkit@its-marketplace
```

Once installed, the skills activate automatically when you write or debug tests.
