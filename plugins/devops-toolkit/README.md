# devops-toolkit

The ITS toolkit for infrastructure, databases and safe deployments. Bundles three skills covering containers, PostgreSQL and rollout planning.

## Skills

| Skill | What it does |
| --- | --- |
| `docker-patterns` | Docker and Docker Compose patterns for local development, container security, networking, volumes and multi-service orchestration. |
| `postgres-patterns` | PostgreSQL patterns for query optimization, schema design, indexing and security. |
| `devops-rollout-plan` | Generate rollout plans with preflight checks, step-by-step deployment, verification signals and rollback procedures. |

## Install

```text
/plugin install devops-toolkit@its-marketplace
```

Once installed, the skills activate automatically when you work on infrastructure, databases or deployments.
