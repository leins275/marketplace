# django-toolkit

The ITS standard for building Django services with Claude Code. Bundles three skills covering how the team architects, secures and verifies Django applications.

## Skills

| Skill | What it does |
| --- | --- |
| `django-patterns` | Django architecture patterns, DRF API design, ORM best practices, caching, signals and middleware for production-grade apps. |
| `django-security` | Authentication, authorization, CSRF protection, SQL-injection and XSS prevention, and secure deployment configuration. |
| `django-verification` | Pre-release verification loop: migrations, linting, tests with coverage, security scans and deployment-readiness checks. |

## Install

```text
/plugin install django-toolkit@its-marketplace
```

Once installed, the skills activate automatically when you work on Django code.
